It is a basic CNN code applied on CIFAR 10 class dataset. 

CNN Structure: 

4 Layers:

1st and 2nd Layers have 64 filters with 5 by 5 kernel size, max pooling layer of 2 by 2 with stride 2 is applied,
3rd Layer has 384 and 4th Layer has 192 neurons.
